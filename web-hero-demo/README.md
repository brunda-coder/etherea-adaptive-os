# Etherea Web Hero Demo

## Run
```bash
cd web-hero-demo
npm install
npm run dev
```

## What this demonstrates
- **Aurora ring + search bar**: cinematic ambient ring and natural-language search input.
- **Floating panels**: command panel, avatar bubble, controls, memory panel.
- **Avatar speaking pulse**: text + pulsing indicator without image assets.
- **Command palette overlay**: keyboard/UX metaphor for NL command entry.
- **Session memory mock**: local-first style memory timeline panel.
- **Toggles**: autonomy, privacy mode, and focus mode switch controls.

## Mapping to desktop features
- Aurora visuals mirror the desktop Aurora bar/ring canvas.
- Command palette mirrors `CommandPalette` in desktop UI.
- Avatar bubble maps to desktop avatar world dialogue interactions.
- Memory panel maps to session snapshot + privacy settings concept.
- Toggles mirror desktop settings/privacy and autonomy override controls.
